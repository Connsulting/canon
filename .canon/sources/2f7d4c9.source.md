## Problem Statement

## Desired Behavior
Run a named td work session, collect in-review and reviewable items, inspect open/in-progress bug tasks, and document actionable follow-ups. If no review items exist, explicitly record a zero-backlog outcome.

## Scope
In scope:
1. Start review session `td-review-2026-03-07`.
2. Query `in_review`, `reviewable`, open bugs, and in-progress bugs.
3. Confirm whether immediate bug fixes or deferred bug tasks are required.
4. Record session results in Canon artifacts.

Out of scope:
1. Creating synthetic tasks when no issues exist.
2. Implementing unrelated product features.
