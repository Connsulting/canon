# Canon Reset

Add `canon reset <spec-id>` to roll Canon history back to a reference spec.

Behavior:
- Find the newest ledger entry with `<spec-id>`.
- Delete every newer Canon artifact:
  - `.canon/ledger/*.json` entries
  - `.canon/specs/*.spec.md` files
  - `.canon/sources/*.source.md` files
- Keep the reference spec and all older history.
- Missing targeted files are tolerated (best effort). Unexpected filesystem errors fail.
- `state/` is not changed; run `canon render --write` when refreshed output is needed.
